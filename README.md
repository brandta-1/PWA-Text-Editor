# J-A-T-E

## Description
J-A-T-E (Just Another Text Editor), bundled with webpack and served as a PWA (progressive web app)
## Installation
Install packages:
```
npm i
```
Install server and client dependencies:
```
npm run install
```

Build the app and start the server:
```
npm run start
```
## Deployment:
